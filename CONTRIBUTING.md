# Contributing to LensPDF

Thanks for considering a contribution!

The full contributor guide — boundary rule, provenance, local
development workflow, public-API surface, commit/PR style, and license
terms — lives at
[**docs/contributing.md**](./docs/contributing.md).

Security issues should follow the disclosure process in
[SECURITY.md](./SECURITY.md), not GitHub issues. By participating you
agree to abide by the [Code of Conduct](./CODE_OF_CONDUCT.md).

are licensed under the same terms.
