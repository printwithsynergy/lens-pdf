# Contributing to LoupePDF

Thanks for considering a contribution!

The full contributor guide — boundary rule, provenance, local
development, public-API surface, commit/PR style, license — lives at
[**docs/contributing.md**](./docs/contributing.md).

For day-to-day:

```sh
npm install
npm run typecheck
npm test
npm run build
```

By participating in this project you agree to abide by the
[Code of Conduct](./CODE_OF_CONDUCT.md). Security issues should follow
the disclosure process in [SECURITY.md](./SECURITY.md), not GitHub
issues.
